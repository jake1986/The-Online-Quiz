Timed Code Quiz


On the following pages, you will be presented with series of questions. You will be alotted 5 minutes to complete the quiz. Read the question and then select the best answer by selecting the button that corresponds to the answer. Then select the answer and then the "Next Question" button in order to move onto the following question. Once a question is answered, "Correct" or "Incorrect" will be displayed before moving on to the next question. If you miss a question, time will be deducted from the timer.
